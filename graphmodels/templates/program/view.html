The correlation model estimates a sparse inverse covariance matrix using a thresholding correlation matrix.

Options:

-t THRESH, --thresh=THRESH
threshold for screening the correlation matrix

Reference:

  • A. O. Hero III and B. Rajaratnam. Large Scale Correlation Screening. ArXiv e-prints, February 2011.
The Glasso model estimates a sparse inverse covariance matrix using a lasso(L1) penalty.

Options:

-l LAMBDA, --lambda=LAMBDA
parameter to control the regularization

Reference:

  • J. Friedman, T. Hastie, and R. Tibshirani, “Sparse inverse covariance estimation with the graphical lasso,” Biostatistics, vol. 9, no. 3, p. 432, 2008.
The CLIME model applies a constrained l1 minimization approach to sparse precision matrix estimation.

Options:

-l LAMBDA, --lambda=LAMBDA
parameter to control the regularization

Reference:

  • Cai, T., Liu, W. and Luo, X. A Constrained L1 Minimization Approach to Sparse Precision Matrix Estimation. Journal of the American Statistical Association 106 594–607, 201.1
The TIGER model applies a Tuning-Insensitive approach for estimating a sparse precision matrix.

Options:

-l LAMBDA, --lambda=LAMBDA
parameter to control the regularization

Reference:

  • Han Liu, Lie Wang. TIGER: A Tuning-Insensitive Approach for Optimally Estimating Gaussian Graphical Models. arXiv:1209.2437, 2012.
